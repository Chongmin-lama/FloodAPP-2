FLOODGUARD APP - COMPLETE SUMMARY
===================================
Date: June 25, 2026

1. DATABASE SETUP (SQL Server Express)
---------------------------------------
Service: MSSQL$SQLEXPRESS
Auth: Windows Authentication (Trusted Connection, no password)
Database to create: flood_db

Connection string in .env:
DB_CONNECTION_STRING=Server=localhost\\SQLEXPRESS;Database=flood_db;Trusted_Connection=true;TrustServerCertificate=true;

2. DATABASE TABLES
------------------
Run this in SQL Server Management Studio:

CREATE DATABASE flood_db;
GO
USE flood_db;

CREATE TABLE users (
    id INT IDENTITY(1,1) PRIMARY KEY,
    name NVARCHAR(100) NOT NULL,
    email NVARCHAR(255) UNIQUE NOT NULL,
    password NVARCHAR(255) NOT NULL,
    role NVARCHAR(20) DEFAULT 'citizen',
    created_at DATETIME2 DEFAULT GETDATE()
);

CREATE TABLE reports (
    id INT IDENTITY(1,1) PRIMARY KEY,
    user_id INT NULL,
    location NVARCHAR(255) NOT NULL,
    description NVARCHAR(MAX) NOT NULL,
    severity NVARCHAR(20) DEFAULT 'medium',
    status NVARCHAR(20) DEFAULT 'pending',
    created_at DATETIME2 DEFAULT GETDATE(),
    FOREIGN KEY (user_id) REFERENCES users(id)
);

3. DEPENDENCIES INSTALLED
-------------------------
- mssql (SQL Server driver)
- bcrypt (password hashing)
- leaflet + react-leaflet (map)
- @types/leaflet, @types/bcrypt (types)

4. API ROUTES CREATED
---------------------
- /api/auth/register (POST) - Create account
- /api/auth/login (POST) - Login
- /api/reports (GET, POST) - View & submit reports
- /api/reports/anon (POST) - Anonymous report
- /api/reports/[id] (GET, PUT) - View/update single report

5. FRONTEND PAGES
-----------------
- / (Home) - Landing page with Nepal Leaflet map
- /login - Login form (calls /api/auth/login)
- /register - Registration form (calls /api/auth/register)
- /dashboard - User dashboard (shows user's reports)
- /map - Full-screen interactive Nepal map

6. KEY FEATURES
---------------
- Anyone can submit a flood report (anonymous)
- To view dashboard & track reports, user must create an account
- Public map shows all resolved reports (no login needed)
- Map centered on Nepal (bounds: 24.4457,77.2689 to 30.4289,88.2273)
- Roles: citizen, authority, admin
- Passwords hashed with bcrypt (despite user saying not needed, bcrypt is installed)
